Acerte o Número Secreto !

O objetivo do jogo é descobrir qual é o número secreto.

(O número está entre 0 e 100)

Regras:

  • O jogo começa assim que algum dos jogadores falar algo.
  • Os jogadores devem dizer números dentro do intervalo de 0 e 100
  • Caso algum jogador falar um número fora do intervalo, o jogo retornará um erro.
  • Caso algum jogador falar uma palavra que não seja um número, o jogo retornará um erro.
  • O jogo acaba assim que algum dos jogadores acertar o número secreto ou se falarem Game Over.
